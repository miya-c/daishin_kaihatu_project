## 1. システム概要

「水道検針システム」は、アパートやマンションなどの集合住宅における水道メーター検針作業を効率化するためのWebアプリケーションです。

### 🏗️ システム構成

- **フロントエンド**: PWA（Progressive Web App）対応のWebアプリ
- **バックエンド**: Google Apps Script（GAS）+ Google スプレッドシート
- **データ管理**: CSVファイル形式でのデータ管理

### 📱 主な機能

1. **物件選択**: 検針対象の物件を選択
2. **部屋選択**: 選択した物件内の部屋一覧表示と検針対象選択
3. **検針入力**: 水道メーター値の入力と自動計算
4. **管理機能**: データ管理、品質チェック、システム診断

## 2. 基本操作手順

### 📱 検針作業の流れ

#### ステップ1: アプリへのアクセス

1. **Webブラウザでアクセス**
   - システムのURLにアクセス
   - PWAとしてインストール可能（ホーム画面に追加）

2. **自動リダイレクト**
   - PWAモードで起動時は自動的に物件選択画面へ移動

#### ステップ2: 物件選択

1. **物件一覧画面**
   ```
   📍 利用可能な物件が一覧表示されます
   ・パルハイツ平田
   ・コーポひろた
   ・コーポ吉嶺
   ・ユートピア与儀
   ・岸本アパート  など
   ```

2. **物件選択**
   - 検針したい物件をタップ/クリック
   - 物件名が大きく表示されるため見やすい

#### ステップ3: 部屋選択

1. **部屋一覧表示**
   ```
   🏠 選択した物件の部屋が表示されます
   例：パルハイツ平田の場合
   ・101号室
   ・201号室  
   ・202号室
   ・301号室  など
   ```

2. **部屋の状態表示**
   - **白色の部屋**: 未検針の状態（検針可能）
   - **グレーの部屋**: 「検針不要」設定（選択不可）
   - **完了マーク付き**: 検針済み（再検針可能）

3. **部屋選択**
   - 検針したい部屋をタップ/クリック

#### ステップ4: 検針値入力

1. **現在値入力**
   ```
   📊 水道メーター読み取り画面
   ・現在の指示数を入力
   ・前回値、前々回値が自動表示
   ・使用量が自動計算
   ・警告判定が自動実行
   ```

2. **データ確認**
   - 入力値の確認
   - 警告フラグの確認
   - 標準偏差による異常値検知

3. **保存**
   - 「保存」ボタンで検針データを記録
   - 自動的に部屋選択画面に戻る

#### ステップ5: 検針完了

1. **物件検針完了**
   - すべての部屋の検針が完了したら
   - 「この物件の検針を完了する」ボタンをクリック
   - 物件マスタに完了日が記録される

### 🔧 実際の操作手順

#### データ管理の実行例

1. **Googleスプレッドシートを開く**
2. **メニューバーの「🔧 水道検針システム」をクリック**
3. **「📊 データ管理」→「1. 物件マスタの物件IDフォーマット」を選択**
4. **確認ダイアログで「はい」をクリック**
5. **処理完了メッセージを確認**

#### 定期メンテナンスの推奨手順

```
月次作業（毎月末）:
1. 📊 データ管理 → 6. 月次検針データ保存とリセット
2. 🔍 データ品質管理 → 1. 重複データクリーンアップ
3. 🔍 データ品質管理 → 2. データ整合性チェック

週次作業（毎週）:
1. ⚙️ システム管理 → 2. システム診断
2. 🔍 データ品質管理 → 3. データ高速検索インデックス作成

随時作業:
- エラー発生時: ⚙️ システム管理 → 3. エラーログ収集
```

## 4. トラブルシューティング

### 🚨 よくある問題と解決方法

#### 4.1 アプリアクセス関連

| 問題 | 原因 | 解決方法 |
|------|------|----------|
| アプリにアクセスできない | URLが間違っている | 管理者にURLを確認 |
| 画面が真っ白 | キャッシュの問題 | ブラウザキャッシュをクリア |
| PWAインストールできない | ブラウザが対応していない | Chrome/Safari等の対応ブラウザを使用 |

#### 4.2 検針作業関連

| 問題 | 原因 | 解決方法 |
|------|------|----------|
| 物件が表示されない | データの問題 | 管理者に連絡 |
| 部屋がグレーアウト | 検針不要設定 | 正常な動作（スキップ対象） |
| 検針値が保存されない | ネットワークエラー | 接続確認後再試行 |
| 警告フラグが表示される | 異常値の可能性 | メーター値を再確認 |

#### 4.3 管理者向け問題

| 問題 | 原因 | 解決方法 |
|------|------|----------|
| メニューが表示されない | スクリプト権限の問題 | GAS実行権限を確認 |
| 処理が途中で止まる | タイムアウト | データ量を減らして再実行 |
| エラーメッセージが出る | データ形式の問題 | データ整合性チェックを実行 |

## 5. よくある質問

### 🤔 一般利用者向けFAQ

**Q1: 検針値を間違えて入力してしまいました。修正できますか？**
A1: はい、同じ部屋を再度選択して正しい値を入力し直してください。最新の値で上書き保存されます。

**Q2: 「検針不要」と表示される部屋があるのはなぜですか？**
A2: 空室や設備上の理由で検針が不要な部屋です。グレーアウト表示され、選択できません。

**Q3: 警告フラグが出た場合はどうすればいいですか？**
A3: メーター値を再確認してください。大幅な使用量の変化がある場合に表示されます。正しい値であればそのまま保存してください。

**Q4: アプリがオフラインでも使えますか？**
A4: PWAとしてインストールすれば一部機能はオフラインでも利用可能ですが、データ保存にはインターネット接続が必要です。

### 🔧 管理者向けFAQ

**Q5: 新しい物件を追加するにはどうすればいいですか？**
A5: 物件マスタシートに新しい物件情報を追加し、部屋マスタに対応する部屋を追加してください。その後「初期検針データ作成」を実行してください。

**Q6: データのバックアップはどのように取ればいいですか？**
A6: Googleスプレッドシート機能でファイルをダウンロードするか、「月次検針データ保存とリセット」機能を使用してください。

**Q7: システムの動作が遅い場合はどうすればいいですか？**
A7: 「データ高速検索インデックス作成」と「全体最適化バッチ処理」を実行してください。

**Q8: 部屋IDの形式を変更したい場合は？**
A8: 「generateRoomIds」機能で物件別にR001, R002...の形式で自動生成できます。
