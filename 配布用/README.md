# 水道検針管理システム - セットアップガイド

## 配布物

このパッケージに含まれるファイル:

```
├── README.md              ← このファイル（セットアップ手順）
└── library_client.gs      ← GASプロジェクトに貼り付けるスクリプト
```

> **注意**: ライブラリ本体はGASライブラリとして開発者が管理しています。
> 各社のプロジェクトからライブラリを参照するため、ソースコードの配布は不要です。

---

## セットアップ手順

### 手順1: Google Spreadsheet の作成

1. Googleアカウントで新規スプレッドシートを作成
2. 名前を「水道検針データ」等に変更

### 手順2: GASプロジェクトの作成

1. スプレッドシートの **「拡張機能 → Apps Script」** を開く
2. デフォルトの「コード.gs」の内容を**全て削除**
3. `library_client.gs` の内容を**コピー＆ペースト**
4. **保存**（Ctrl+S）

### 手順3: ライブラリの追加

1. 左サイドバーの **「ライブラリ」** → **「ライブラリを追加（+）」**
2. **スクリプトID** に以下を入力:
   ```
   1iSR_dY3jWIN3rXRXMeYKyulOggtmM2LDhkzlFqM9MccWCupqEwquE04-
   ```
3. **識別子** を `cmlibrary` に設定
4. バージョンは **「HEAD（開発モード）」** を選択して **「追加」**

> **重要**: HEADを選択することで、開発者側のライブラリ更新が自動的に反映されます。特定バージョンを選択しないでください。

### 手順4: スクリプトプロパティの設定

プロジェクト設定（歯車アイコン）→ スクリプトプロパティに以下を追加:

| プロパティ名 | 値 | 例 |
|---|---|---|
| `API_KEY` | `sk-` + ランダム文字列 | `sk-a1b2c3d4e5f6g7h8` |
| `ADMIN_TOKEN` | `admin-` + ランダム文字列 | `admin-x7y8z9w0v1u2t3` |

> API_KEYとADMIN_TOKENは推測困難な文字列を設定してください。

### 手順5: Web App のデプロイ

1. **「デプロイ」** → **「新しいデプロイ」**
2. 種類: **ウェブアプリ**
3. 実行するバージョン: **新規**
4. 実行 as: **自分**
5. アクセスできるユーザー: **全員**
6. デプロイ → **URLをコピー**（次の手順で使用）

### 手順6: 初期セットアップ

スプレッドシートに戻り、メニューから **「🔧 システム設定 → 導入ウィザード」** を実行:

| ステップ | 内容 |
|---|---|
| 1. システム診断 | 環境の確認 |
| 2. マスタシート作成 | 必要なシートの自動生成 |
| 3. サンプルデータ投入 | テスト用データ（スキップ可） |
| 4. 物件ID割り当て | 物件IDのフォーマット |
| 5. 部屋ID生成 | 部屋IDの自動生成 |
| 6. 検針データ作成 | 検針データシートの初期化 |
| 7. 最終確認 | 動作テスト |

### 手順7: 動作確認

- [ ] ブラウザで `{GAS_URL}?action=test` → 接続テスト成功
- [ ] 管理画面（`{GAS_URL}`）→ ADMIN_TOKEN でログイン → ダッシュボード表示
- [ ] 管理画面で物件を1件追加 → スプレッドシートに反映
- [ ] 管理画面で部屋を追加 → スプレッドシートに反映

## 管理画面へのアクセス

Web App URL（手順5でコピーしたURL）にブラウザでアクセス → ADMIN_TOKENを入力してログイン。

管理画面で以下の操作が可能:
- 物件・部屋の追加・編集・削除
- 検針状況の確認（一括表示・年間レポート）
- 月次処理（データ保存とリセット）
- データ検証・クリーンアップ

## PWA設定（検針員用スマホアプリ）

### 初回設定

検針員がスマホでPWAにアクセスすると、初期設定画面が表示されます。

1. ブラウザで PWA URL にアクセス
2. **GAS Web App URL**（手順5で取得したURL）を入力
3. **APIキー**（手順4で設定したAPI_KEY）を入力
4. **「設定して開始」** をタップ → 物件選択画面に遷移

### 設定リンクの共有（推奨）

設定情報をURLに埋め込んだリンクを生成して、検針員に共有できます。

1. PWAの **⚙️ 設定** → **「リンクを生成」** をタップ
2. 生成されたURLをコピーして検針員に送信
3. 検針員がリンクを開くと設定が自動完了

リンクの形式:
```
https://（PWA URL）/#url=https%3A%2F%2Fscript.google.com%2F...&key=API_KEY
```

> **注意**: 設定リンクにはAPI_KEYが含まれます。取り扱いにご注意ください。

### 設定の変更・削除

- PWA内の **⚙️ 設定** から接続設定の編集・削除が可能です
- 設定を削除すると初期設定画面に戻ります

### ホーム画面に追加

PWAはスマホのホーム画面に追加してアプリとして使用できます:
- **iOS**: Safariで共有ボタン → 「ホーム画面に追加」
- **Android**: Chromeでメニュー → 「ホーム画面に追加」

## トラブルシューティング

| 問題 | 対処 |
|---|---|
| 「cmlibrary is not defined」エラー | 手順3でライブラリが正しく追加されているか確認（識別子が `cmlibrary` であること） |
| API_KEY エラー | 手順4のスクリプトプロパティが正しく設定されているか確認 |
| 管理画面が表示されない | 手順5でWeb Appがデプロイされているか確認 |
| シートが存在しないエラー | 手順6の導入ウィザードを実行 |
| メニューが表示されない | スプレッドシートを開き直す（メニューがリフレッシュされます） |

---

## ライブラリの更新

ライブラリは開発者側で一元管理されています。開発者がライブラリを更新すると、
各社のプロジェクトは**次回実行時に自動的に最新版**を使用します。

更新が反映されない場合は:
1. GASエディタでライブラリのバージョンを確認
2. 最新版に更新して保存
